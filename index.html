🏷️ HTMLタグの分類（基礎と応用）

🔰 基礎タグ

  • <h1>〜<h6>：見出し
  • <p>：段落（文章）
  • <img>：画像
  • <a>：リンク
  • <ul>/<li>：リスト

📚 応用タグ

  • <section>：意味あるかたまり（セクション）
  • <article>：記事やブログなど
  • <nav>：ナビゲーションメニュー
  • <header>/<footer>：ページの上と下
  • <figure>/<figcaption>：画像＋キャプション

📐 レイアウトのやり方と仕組み

Webの要素はすべて「ボックス」として扱われます。

  • display: block / inline / flex / grid
  • margin: 要素の外側の余白
  • padding: 要素の内側の余白
  • border: 要素の枠線

よく使うレイアウト手法：

  1. Flexbox：横並び、縦並びなどに便利
  2. Grid：複雑な配置も表形式で調整可能
  3. position: relative / absolute / fixed で自由配置
