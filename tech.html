🌐 HTML / CSS / JSとは？

  • HTML： ページの構造をつくる言語
  • CSS： 色・配置などデザインを担当
  • JavaScript： 動きを加える（例：クリック時に画像が変わる）

⚡ 効率アップのテクニック

  • ✅ エディタはVS Code：拡張機能が豊富、初心者も安心
  • ✅ Emmet記法：HTMLを短く書ける（例：ul>li*3 → 3つのリスト）
  • ✅ ライブプレビュー：拡張機能「Live Server」でブラウザ確認が一瞬
  • ✅ テンプレート活用：HTML5テンプレートやCSSリセットを使おう
  • ✅ ショートカットキー：コピー・貼り付け、複製など時短の鍵
